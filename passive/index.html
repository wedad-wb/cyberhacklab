Lab 1

Search for Passive Information Gathering

You can start with Whois, crt.sh, MXToolbox, theHarvester, Google Dorks, and explore many other OSINT tools without touching the target servers.

Now continue your exploration using the following tools:

DNSDumpster: Discover subdomains, IPs, and DNS servers visually.

Netcraft: Analyze hosting info, certificates, uptime, and technologies used.

Shodan: Search for exposed devices, services, and open ports across the internet.

VirusTotal: Investigate domain reputation, DNS history, and linked IPs.

BuiltWith: Uncover technologies used in building the target site (servers, CMS, JS...).
